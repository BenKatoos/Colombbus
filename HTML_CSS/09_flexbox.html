C | Direction

Flexbox nous permet d'agencer ces éléments dans le sens que l'on veut. Avec flex-direction, on peut les positionné aussi bien horizontalement que verticalement, ou encore les inversés. Il peut prendre les valeurs suivante :
-> row : organisé sur une ligne (par défaut)
-> column : organisé sur une colonne
-> row-reverse : organisé sur une ligne en ordre inversé
-> column-reverse : organisé sur une colonne en ordre inversé

D | Retour à la ligne

Par défaut les blocs essaient de rester sur la même ligne s'ils n'ont pas la place (ce qui provoque des bugs de design parfois) Si vous voulez vous pouvez demandr à ce que les blocs aillent à la ligne lorsqu'ils n'ont plus la place avec flex-wrap qui peut prendre ces valeurs :
-> - nowrap
-> - wrap : les éléments vont à la ligne quand il n'y a plus de place
-> - wrap-reverse : les éléments vont à la ligne quand il n'y a plus de place mais de façon inversé

E | Alignement

Les éléments sont organisé soit horizontalement (par défaut) soit verticalement. Cela définit ce qu'on appelle l'axe principal. Il y'a aussi un axe secondaire (cross axis) :
-> si ces éléments sont organisé horizontalement l'axe secondaire est l'axe vertical
-> si ces éléments sont organisé verticalement l'axe secondaire est l'axe horizontal

Alignement sur l'axe principal

Afin d'aligner nos éléments sur l'axe principal :
justify-content
elle peut prendre comme valeur :
-> flex-start : aligner au début (par défaut)
-> flex-end : aligner à la fin
-> center : aligner au centre
-> space-between : les éléments sont étirés sur tout l'axe (il y'a de l'espace entre eux)
-> space-around : les éléments sont étirés sur tout l'axe mais il laisse aussi de l'espace sur les extrémitées
-> space-evenly: les élments sont répartis uniformément dnas le conteneur d'alignement le lonf de l'axe principal.

F | Répartir plusieurs lignes

Si vous avez plusieurs lignes dans votre flex-box vous pouvez choisir comment celle-ci seront réparties avec align-content Cette propriété permet de gerer l'alignement des différents lignes (ou colonnes) les par rapports au autres. Cette propriété n'aura d'effet que si le conteneur flexible possède plusieurs lignes ( si l'axe principal est horizontal) ou plusieurs colonnes (si l'axe principalest est vertical) et que si le flex-wrap est présent. Les valeurs de cette propriété :
-> flex-start : les éléments sont placés au début
-> flex-end : les éléments sont placés à la fin
-> center : les éléments sont placé au centre
-> space-between : les éléments sont séparé avec de l'espace entre eux
-> space-around : les éléments sont séparé avec de l'espace entre eux mais il y a aussi de l'espace au début ainsi qu'à la fin
-> space-evenly: les élments sont répartis uniformément dnas le conteneur d'alignement le lonf de l'axe principal. -> stretch : (valeur par défaut) les éléments s'étirent afin d'occuper tout l'espace

Stretch -> par défaut
